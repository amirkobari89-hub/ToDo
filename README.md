# برنامه مدیریت لیست کارها (Todo)

این پروژه یک برنامه کنسولی ساده با زبان Java است که برای تمرین مفاهیم پایه برنامه‌نویسی و OOP طراحی شده است.

## قابلیت‌ها
- افزودن کار جدید با عنوان و اولویت (1 تا 3)
- نمایش فقط کارهای انجام‌نشده
- تغییر وضعیت کار (انجام شده / انجام نشده)
- حذف کار بر اساس شماره
- منوی فارسی

## ساختار پروژه
پروژه شامل دو کلاس اصلی است:

### 1. کلاس Task
نمایانگر یک کار است و شامل:
- عنوان کار
- اولویت
- وضعیت انجام شدن

این کلاس از اصل Encapsulation استفاده می‌کند و فیلدها به صورت private تعریف شده‌اند.

### 2. کلاس TodoAppSimple
کلاس اصلی برنامه که:
- منوی برنامه را نمایش می‌دهد
- ورودی کاربر را دریافت می‌کند
- لیست کارها را با ArrayList مدیریت می‌کند

## مفاهیم استفاده شده
- class و object
- ArrayList
- if / else
- boolean
- encapsulation
- constructor
- Scanner برای ورودی

## نحوه اجرا
1. فایل‌ها را در یک پروژه Java قرار دهید
2. کلاس TodoAppSimple را اجرا کنید
3. از منوی برنامه استفاده کنید

این پروژه برای افراد تازه‌کار مناسب است و هدف آن یادگیری مفاهیم پایه جاوا می‌باشد.
